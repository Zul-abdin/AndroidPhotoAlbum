How to use Android Photos App

Album Activity:
    + FAB: Create new Album
    Search FAB: Search for photos by tags
    Hold an album to rename
    Swipe album left/right to delete
    Click an album to go to Gallery Activity

Gallery Activity:
    + FAB: Add new photo to album
    Hold photo to access image options to delete/move it
    Images can also be deleted by swiping left/right
    Click a photo to go to Photo Activity

Photo Activity:
    + FAB: Add new tag to photo
    Enter Slideshow Button: Enter Slideshow Activity
    Hold tag to edit
    Swipe tag left/right to delete

Slideshow Activity
    Next: Display next image in album
    Prev: Display previous image in album

To go to previous activity, click the back button on android home bar.
